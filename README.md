# Personal Portfolio Website

This is my personal developer portfolio built as part of **Future Interns – Full Stack Web Development Internship (Task 1)**.  
It showcases my skills, projects, and contact information in a modern responsive interface.

## Features
- Responsive portfolio website
- Built with React and Tailwind CSS
- Projects showcase section
- About section
- Contact form with Formspree
- Modern UI design

## Tech Stack
- React.js
- Tailwind CSS
- JavaScript
- HTML / CSS
- Vite
